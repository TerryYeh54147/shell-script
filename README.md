# shell-script

## get_long_run_log.sh

find out wich log file is long run job

```txt
get_long_run_log.sh options:
        -f|--file_path:
            target file path
        -d|--directory_path:
            [optoin] log directory path(default: ./)
        -t|--limited_time:
            [optoin] job limited runtime(default: ./)
        -h|--help:
            show all options detail
```
